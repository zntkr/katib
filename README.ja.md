# Katib

**Windows 向けオフライン音声入力。キーを押しながら話して、離すだけ — カーソル位置にテキストが入力されます。**

クラウド不要。サブスクリプション不要。データは一切外部に送信されません。

## 使い方

1. 任意のテキストフィールドをクリック — ブラウザ、Word、IDE、チャットなど、Windows 上のどこでも
2. ホットキー（デフォルト **F9**）を押しながら話し、離す
3. 音声は [Whisper](https://github.com/openai/whisper) によってローカルで文字起こしされ、カーソル位置に入力されます

モデルの初回ダウンロード後はインターネット接続不要です。

---

## 機能

- **完全オフライン** — Whisper は CPU 上で完全動作。API キー不要、アカウント不要
- **あらゆる場所で使用可能** — Windows アプリケーションの任意のテキストフィールドに対応
- **プッシュトゥトーク** — 押して録音、離して文字起こし。誤作動なし
- **5 種類のモデルサイズ** — 150 MB から 3 GB まで。ハードウェアに合わせて速度と精度を選択
- **18 の文字起こし言語** — アラビア語、中国語、英語、フランス語、ドイツ語、ギリシャ語、ヒンディー語、インドネシア語、イタリア語、日本語、韓国語、ペルシア語、ポルトガル語、ロシア語、スペイン語、トルコ語、ウルドゥー語、および自動検出
- **11 の UI 言語** — English、Türkçe、Deutsch、Español、Français、Português、Русский、日本語、中文、한국어、العربية — 完全 RTL レイアウト対応
- **言語別 AI プロンプト** — 音声言語ごとにコンテキストを設定。切り替え時に自動読み込み
- **テレメトリーなし** — 分析データ、クラッシュレポート、アップデート通知は一切なし

## インストール

### オプション A — ビルド済みバイナリ（推奨）

[Releases](../../releases) ページから最新リリースをダウンロードし、解凍後に `Katib.exe` を実行してください。

> バイナリは署名されていないため、**Windows SmartScreen** が警告を表示する場合があります。**詳細情報 → 実行** をクリックしてください。

### オプション B — ソースから実行

Python 3.10 以上が必要です

```bash
git clone https://github.com/zntkr/katib.git
cd katib
pip install -r requirements.txt
python main.py
```

## モデル

| モデル | サイズ | 速度 | 精度 | 備考 |
|--------|--------|------|------|------|
| tiny | ~150 MB | 非常に速い | 低 | 低スペックまたは旧式のハードウェア向け |
| base | ~300 MB | 速い | 普通 | |
| **small** | **~500 MB** | **バランス型** | **良好** | **推奨の出発点** |
| medium | ~1.5 GB | 遅い | 高 | 専門用語や訛りのある音声に対応 |
| large-v3 | ~3 GB | 非常に遅い | 最高 | 最高精度 |

モデルは [Hugging Face (Systran)](https://huggingface.co/Systran) から取得し、ローカルに保存されます。既にダウンロード済みのモデルが入ったフォルダを Katib に指定することも可能です。

## 設定

| 設定項目 | 説明 |
|----------|------|
| **ホットキー** | 任意のキーまたはキーの組み合わせ。デフォルト: F9 |
| **音声言語** | 特定の言語を指定するか、自動検出を使用 |
| **精度** | `int8`（デフォルト）· `int8_float32` · `float32` — 速度と精度のトレードオフ |
| **AI プロンプト** | 文字起こしごとに Whisper に渡すコンテキスト。音声言語ごとに個別保存 |
| **アプリ言語** | UI の表示言語 — 即時変更、再起動不要 |

## プライバシー

Katib はエアギャップ環境でも使用できるよう設計されています：

- 実行中の外部通信なし
- 使用統計、エラーレポート、アップデート確認なし
- 設定とログは `%LOCALAPPDATA%\Katib` にローカル保存
- ネットワーク通信が発生するのは、オプションの初回モデルダウンロード時のみ
