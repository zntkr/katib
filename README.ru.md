# Katib

**Офлайн-диктовка для Windows. Зажмите клавишу, говорите, отпустите — текст появляется в курсоре.**

Без облака. Без подписки. Ничего не покидает ваш компьютер.

## Как это работает

1. Нажмите в любое текстовое поле — браузер, Word, IDE, чат, в любом месте Windows
2. Зажмите горячую клавишу (по умолчанию **F9**), говорите, отпустите
3. Ваши слова транскрибируются локально с помощью [Whisper](https://github.com/openai/whisper) и вводятся в курсоре

После однократной загрузки модели интернет не требуется.

---

## Возможности

- **100% офлайн** — Whisper работает полностью на вашем CPU, без API-ключей и учётных записей
- **Работает везде** — любое фокусируемое текстовое поле в любом приложении Windows
- **Push-to-talk** — зажмите для записи, отпустите для транскрипции; без случайных срабатываний
- **Пять размеров модели** — от 150 МБ до 3 ГБ; выбирайте баланс скорости и точности под своё железо
- **18 языков транскрипции** — арабский, китайский, английский, французский, немецкий, греческий, хинди, индонезийский, итальянский, японский, корейский, персидский, португальский, русский, испанский, турецкий, урду и автоопределение
- **11 языков интерфейса** — English, Türkçe, Deutsch, Español, Français, Português, Русский, 日本語, 中文, 한국어, العربية — с полной поддержкой RTL-раскладки
- **Промпты ИИ по языкам** — задайте контекст для каждого языка речи; загружается автоматически при переключении
- **Без телеметрии** — никакой аналитики, отчётов об ошибках или проверок обновлений

## Установка

### Вариант A — Готовый бинарник (рекомендуется)

Скачайте последний релиз со страницы [Releases](../../releases), распакуйте и запустите `Katib.exe`.

> **Windows SmartScreen** может показать предупреждение, так как файл не подписан. Нажмите **Подробнее → Выполнить в любом случае**.

### Вариант B — Из исходного кода

Требуется Python 3.10+

```bash
git clone https://github.com/zntkr/katib.git
cd katib
pip install -r requirements.txt
python main.py
```

## Модели

| Модель | Размер | Скорость | Точность | Примечания |
|--------|--------|----------|----------|------------|
| tiny | ~150 МБ | Очень высокая | Низкая | Бюджетное или устаревшее железо |
| base | ~300 МБ | Высокая | Средняя | |
| **small** | **~500 МБ** | **Сбалансированная** | **Хорошая** | **Рекомендуется для начала** |
| medium | ~1,5 ГБ | Низкая | Высокая | Технический контент, акценты |
| large-v3 | ~3 ГБ | Очень низкая | Максимальная | Наилучшая точность |

Модели загружаются с [Hugging Face (Systran)](https://huggingface.co/Systran) и хранятся локально. Также можно указать Katib папку с уже скачанной моделью.

## Настройки

| Настройка | Описание |
|-----------|----------|
| **Hotkey** | Любая клавиша или комбинация. По умолчанию: F9 |
| **Язык речи** | Принудительно выбрать язык или использовать автоопределение |
| **Точность** | `int8` (по умолчанию) · `int8_float32` · `float32` — скорость vs. точность |
| **Промпт ИИ** | Контекст, передаваемый Whisper при каждой транскрипции; сохраняется отдельно для каждого языка речи |
| **Язык интерфейса** | Язык UI — меняется мгновенно, без перезапуска |

## Конфиденциальность

Katib разработан с расчётом на изолированные среды:

- Никаких исходящих соединений во время работы
- Никакой статистики использования, отчётов об ошибках или проверок обновлений
- Настройки и логи хранятся локально в `%LOCALAPPDATA%\Katib`
- Единственная сетевая активность — однократная и необязательная загрузка модели
