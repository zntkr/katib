# Triage Labels

The following labels are used to manage the lifecycle of issues in this repository.

| Role | Label | Description |
|---|---|---|
| `needs-triage` | `needs-triage` | Newly created issues that need evaluation. |
| `needs-info` | `needs-info` | Issues waiting for more information from the reporter. |
| `ready-for-agent` | `ready-for-agent` | Fully specified issues ready for an AI agent to implement. |
| `ready-for-human` | `ready-for-human` | Issues that require human intervention or implementation. |
| `wontfix` | `wontfix` | Issues that will not be addressed. |
