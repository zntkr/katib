# Katib

**إملاء صوتي بلا إنترنت لنظام Windows. اضغط على المفتاح، تحدّث، أفلته — يظهر النص عند المؤشر.**

بلا سحابة. بلا اشتراك. لا يغادر أي شيء جهازك.

## كيف يعمل

1. انقر على أي حقل نصي — متصفح، Word، بيئة تطوير، دردشة، أي مكان في Windows
2. اضغط مع الاستمرار على مفتاح الاختصار (الافتراضي **F9**)، تحدّث، ثم أفلته
3. يحوّل [Whisper](https://github.com/openai/whisper) كلامك نصًا محليًا ويكتبه عند موضع المؤشر

لا حاجة للإنترنت بعد تنزيل النموذج مرة واحدة.

---

## المميزات

- **100% بلا إنترنت** — يعمل Whisper كليًا على CPU دون الحاجة إلى مفاتيح API أو حسابات
- **يعمل في كل مكان** — أي حقل نصي قابل للتركيز في أي تطبيق Windows
- **الضغط للتحدث** — اضغط للتسجيل، أفلت للتحويل؛ لا تشغيل عرضي
- **خمسة أحجام للنماذج** — من 150 MB إلى 3 GB؛ وازن بين السرعة والدقة حسب جهازك
- **18 لغة للنسخ** — العربية، الصينية، الإنجليزية، الفرنسية، الألمانية، اليونانية، الهندية، الإندونيسية، الإيطالية، اليابانية، الكورية، الفارسية، البرتغالية، الروسية، الإسبانية، التركية، الأردية، والكشف التلقائي
- **11 لغة للواجهة** — English, Türkçe, Deutsch, Español, Français, Português, Русский, 日本語, 中文, 한국어, العربية — مع دعم كامل لتخطيط RTL
- **مطالبات AI لكل لغة** — اضبط السياق لكل لغة كلام؛ يُحمَّل تلقائيًا عند التبديل
- **بلا قياسات عن بُعد** — لا إحصاءات استخدام، ولا تقارير أعطال، ولا فحوصات تحديث

## التثبيت

### الخيار أ — ملف تنفيذي جاهز (موصى به)

نزّل أحدث إصدار من صفحة [Releases](../../releases)، فك الضغط، ثم شغّل `Katib.exe`.

> قد يعرض **Windows SmartScreen** تحذيرًا لأن الملف غير موقّع رقميًا. انقر على **مزيد من المعلومات ← تشغيل على أي حال**.

### الخيار ب — من المصدر

يتطلب Python 3.10 أو أحدث

```bash
git clone https://github.com/zntkr/katib.git
cd katib
pip install -r requirements.txt
python main.py
```

## النماذج

| النموذج | الحجم | السرعة | الدقة | ملاحظات |
|---------|-------|--------|-------|---------|
| tiny | ~150 MB | سريع جدًا | منخفضة | أجهزة ضعيفة أو قديمة |
| base | ~300 MB | سريع | مقبولة | |
| **small** | **~500 MB** | **متوازن** | **جيدة** | **نقطة البداية الموصى بها** |
| medium | ~1.5 GB | بطيء | عالية | المحتوى التقني والنطق المختلف |
| large-v3 | ~3 GB | بطيء جدًا | الأعلى | أفضل دقة ممكنة |

تُجلب النماذج من [Hugging Face (Systran)](https://huggingface.co/Systran) وتُخزَّن محليًا. يمكنك أيضًا توجيه Katib إلى مجلد يحتوي على نموذج نزّلته مسبقًا.

## الإعدادات

| الإعداد | الوصف |
|---------|-------|
| **مفتاح الاختصار** | أي مفتاح أو تركيبة. الافتراضي: F9 |
| **لغة الكلام** | تحديد لغة معينة أو استخدام الكشف التلقائي |
| **الدقة** | `int8` (الافتراضي) · `int8_float32` · `float32` — سرعة مقابل دقة |
| **مطالبة AI** | سياق يُمرَّر إلى Whisper مع كل نسخ؛ يُحفظ بشكل منفصل لكل لغة كلام |
| **لغة التطبيق** | لغة الواجهة — تتغير فورًا دون الحاجة لإعادة التشغيل |

## الخصوصية

صُمِّم Katib ليعمل في بيئات معزولة تمامًا عن الإنترنت:

- لا اتصالات خارجية أثناء التشغيل
- لا إحصاءات استخدام، ولا تقارير أخطاء، ولا فحوصات تحديث
- تُخزَّن الإعدادات والسجلات محليًا في `%LOCALAPPDATA%\Katib`
- النشاط الشبكي الوحيد هو التنزيل الاختياري للنموذج مرة واحدة
